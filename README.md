# Evergreen Tree Generator

[![Open in MATLAB Online](https://www.mathworks.com/images/responsive/global/open-in-matlab-online.svg)](https://matlab.mathworks.com/open/github/v1?repo=nothans/holiday_tree_maker_matlab)

A MATLAB App Designer application for procedurally generating customizable evergreen trees.

## Features

- Adjustable tree height, layers, and trunk dimensions
- Color themes: Classic, Winter, Autumn, Festive
- Optional decorations: ornaments, star, snow
- Export to image (PNG/JPG/PDF) or standalone MATLAB script

## Usage

```matlab
app = EvergreenTreeGenerator();
```

## Requirements

MATLAB R2019b or later with App Designer support.
